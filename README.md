# Web Scraping & Data Extraction Portfolio

This repository contains a collection of real-world web scraping projects built using Python.

The goal of this repository is to demonstrate practical techniques for extracting, cleaning, and structuring data from websites.

## Technologies

- Python
- BeautifulSoup
- Selenium
- Playwright
- Scrapy
- Requests
- Pandas

## Project Categories

### Healthcare Data Scraping
Scrapers that collect pharmaceutical drug shortage data from European health authority websites.

Countries covered:
- Belgium
- Finland
- Netherlands
- Spain
- Portugal

### Future Categories

This repository will continue expanding with additional scrapers including:

- E-commerce product data
- Job listings
- Real estate data
- News articles
- Public datasets

## Output Formats

The scrapers export structured data in:

- CSV
- JSON
- Database-ready formats

## Skills Demonstrated

- Web scraping automation
- Data extraction pipelines
- Handling dynamic websites
- Pagination scraping
- Multi-country data normalization
- Automation with Selenium and Playwright
