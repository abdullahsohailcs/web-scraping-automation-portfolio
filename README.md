# Web Scraping & Data Extraction Portfolio

Python | Web Scraping | Selenium | Playwright | Scrapy | Data Extraction

A collection of real-world web scraping and automation projects built using Python.
This repository demonstrates practical techniques for extracting structured data from websites using modern scraping frameworks and automation tools.

The projects focus on building reliable scrapers capable of handling static and dynamic web pages, pagination, structured data extraction, and exporting clean datasets for analysis or downstream applications.

---

# Project Overview

This repository contains multiple scraping projects organized by domain. Each project demonstrates different scraping challenges such as:

* Extracting structured data from HTML pages
* Automating browser interaction for dynamic websites
* Handling pagination and large datasets
* Cleaning and normalizing extracted data
* Exporting datasets in machine-readable formats

The repository is designed as a long-term portfolio where additional scraping projects will be added over time.

# Current Projects

## Healthcare Data Scraping

### European Drug Shortage Monitoring

This project collects pharmaceutical drug shortage data from official health authority websites across several European countries.

The scrapers automatically extract structured information about medicines that are temporarily unavailable or experiencing supply disruptions.

### Countries Covered

* Belgium
* Finland
* Netherlands
* Spain
* Portugal

### Example Data Collected

* Drug name
* Active ingredient
* Manufacturer
* Shortage start date
* Expected availability date
* Shortage status
* Country
* Source URL

This data can be useful for:

* healthcare research
* pharmaceutical supply monitoring
* data analysis
* market research

# Installation

Clone the repository:

```
git clone https://github.com/yourusername/web-scraping-portfolio.git
```

Navigate to the project directory:

```
cd web-scraping-portfolio
```

Install dependencies:

```
pip install -r requirements.txt
```

---

# Running a Scraper

Example:

```
python healthcare/drug_shortage_eu/belgium_scraper.py
```

The script will extract data and generate a structured dataset in the output directory.
